# FlowLocal

**Диктовка, которая работает без интернета.** Зажали клавишу — говорите.
Отпустили — текст уже в том окне, где вы и печатали.

```
npx flowlocal
```

Скачает свежий установщик и запустит его. Прав администратора не нужно.

## Что это

Голос не покидает этот компьютер: не потому что мы обещаем, а потому что
некуда. В программе нет ни аккаунта, ни сервера, ни адреса, куда слать.
Интернет нужен дважды — скачать модель и проверить обновления.

Распознаёт **GigaAM v3** от SberDevices: 700 000 часов русской речи, 226 МБ,
на процессоре. По карточке модели — 8.4 % ошибок против 25.1 % у Whisper.

- слова-паразиты убираются без модели и без задержки;
- поправки на ходу: «в пятницу, нет, в субботу» → останется суббота;
- короткие фразы заводятся сами: программа замечает, что вы повторяете;
- правка голосом: выделил, сказал «сделай короче» — заменилось.

Windows 10 или 11. Лицензия MIT.

## Про этот пакет

Он не содержит программу и не является ею. FlowLocal написан на Python, и в
npm ему не место — здесь живут пакеты JavaScript. Пакет делает ровно одно:
приносит установщик из [GitHub Releases](https://github.com/romankandeevy/flowlocal/releases)
и передаёт ему управление. Так же поступают esbuild и playwright со своими
бинарниками.

Дальше npm не нужен вовсе: обновления приходят внутри приложения.

Не хотите через npm — [скачайте установщик напрямую](https://github.com/romankandeevy/flowlocal/releases/latest).
