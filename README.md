# FlowLocal

Приватный локальный аналог Wispr Flow для Windows. Зажал клавишу - говоришь,
отпустил - распознанный текст вставился в активное окно. Аудио и текст не
покидают машину: распознавание - faster-whisper (large-v3-turbo) на GPU/CPU.

## Запуск

```powershell
pip install -r requirements.txt
python app.py            # первый запуск скачает модель (~1.6 ГБ) в models/
```

Фоново без консоли: `pythonw app.py`.
Автозапуск с Windows: `python app.py --autostart on` (выключить: `off`).

## Использование

- Зажать **Right Ctrl** (меняется в `config.json` -> `hotkey`), говорить, отпустить.
- Внизу экрана пилюля: `● запись` -> `…распознаю` -> текст вставлен.
- Иконка в трее: серая - готов, красная - запись, золотая - распознавание.
  Меню: конфиг, история, лог, перечитать конфиг, выход.
- Нажатие короче 0.3 с игнорируется (случайные касания).

## config.json

| Ключ | Что делает |
|---|---|
| `hotkey` | клавиша push-to-talk (одна клавиша, имена - как в библиотеке `keyboard`) |
| `language` | `null` - автоопределение; `"ru"` / `"en"` - жёстко |
| `model` | `large-v3-turbo` (баланс), `medium`, `small` (быстрее/слабее) |
| `device` | `auto`: CUDA -> при ошибке CPU. Можно `"cuda"` / `"cpu"` |
| `keep_mic_open` | поток микрофона всегда открыт + пре-буфер 0.5 с - не режется первое слово |
| `insert_mode` | `paste` (Ctrl+V, мгновенно) или `type` (посимвольно, для cmd.exe) |
| `restore_clipboard` | вернуть прежний текст буфера после вставки |
| `remove_fillers` | вырезать «эм», «ммм», «um» и т.п. |
| `dictionary` | свои слова/имена - подсказки распознаванию (hotwords) |
| `llm.enabled` | полировка текста локальной LLM через Ollama (не обязательно) |
| `history` | писать диктовки в `history.jsonl` |

## Полировка через Ollama (опционально)

```powershell
ollama pull qwen2.5:3b
```

В `config.json`: `"llm": {"enabled": true, ...}`. LLM убирает слова-паразиты и
учитывает самоисправления («в пятницу, нет, в субботу» -> «в субботу»).
Недоступна Ollama - вставится обычная расшифровка, диктовка не ломается.

## Ограничения

- Вставка в режиме `paste` перезаписывает не-текстовое содержимое буфера
  (файлы, картинки) - восстанавливается только текст.
- Хоткей - одна клавиша, без комбинаций.
- В приложениях, запущенных от администратора, вставка не сработает,
  если FlowLocal запущен без прав администратора (ограничение Windows).
